---
Database Tables Used (Supabase)
users - User profiles
clips - Video clips and reels
screenshots - Screenshots
games - Games (from Twitch)
likes - Clip/screenshot likes
comments - Comments on clips/screenshots
reactions - Fire reactions on clips
messages - Direct messages
blocked_users - Block relationships
rewards - Reward definitions
user_rewards - User claimed rewards
asset_rewards - Asset rewards
---
External APIs Used
1. Twitch API - Game search, top games
2. Discord OAuth - Social login
3. Google OAuth - Social login

---
Authentication
JWT-based with access tokens (7 days) and refresh tokens (30 days)
Tokens signed with JWT_SECRET and JWT_REFRESH_SECRET
Bearer token in Authorization header
