Update the global CSS/theme of the entire Gamefolio app to match a modern dark gaming SaaS style with neon green accents and Space Grotesk font.

Goal:
Restyle the entire app UI (without changing layout or functionality) to feel premium, modern, and consistent with a dark gaming aesthetic.

Font:
Use Space Grotesk as the global font.

Import it:
@import url('https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700;800&display=swap');

Apply globally:
body {
  font-family: 'Space Grotesk', sans-serif;
}

If using Tailwind, update config:
fontFamily: {
  sans: ['Space Grotesk', 'sans-serif'],
}

Design instructions:

- Replace all existing colours with this palette
- Use dark backgrounds instead of light themes
- Primary buttons should use neon green (#B7FF1A) with dark text
- Add subtle glow to primary buttons and key CTAs
- Cards should be dark with soft borders and rounded corners
- Use border radius between 14px and 20px
- Use soft shadows (no harsh drop shadows)
- Add hover states:
  - Buttons: slightly brighter green + glow
  - Cards: subtle lift + border highlight
- Add smooth transitions (0.2s–0.3s ease)

Components to update:

- Navbar
- Sidebar
- Buttons (primary, secondary, ghost)
- Cards
- Modals
- Forms (inputs, selects, textareas)
- Dropdowns
- Tabs
- Badges
- Links
- Tables (if any)

Inputs:

- Dark background (#0B1218)

- Border: #1B2A33

Focus states:

Typography:

- Use Space Grotesk for all text
- Headings should be bold and clean
- Increase letter spacing slightly for headings
- Keep body text readable with good contrast

Overall feel:

- Minimal but bold
- Similar to modern SaaS landing pages
- Consistent across all pages
