Standardise the thumbnail/card design across the entire Gamefolio app.

Goal:
Apply the same clean thumbnail style currently used for Reels to Clips as well, across all pages including individual user Gamefolio/profile pages.

Current desired Reels style:
- Thumbnail/video preview is clean
- Only small overlay metadata appears on the thumbnail
- Title, username, and game tag sit underneath the thumbnail
- Neon green game tag
- Minimal clutter
- Premium gaming platform feel

Apply this same design language to:

- Clips grids
- Trending clips
- Explore clips
- Profile/Gamefolio clip sections
- Individual user Gamefolio pages
- Search results
- Any clip thumbnail/card shown across the app

For all Reel and Clip cards:

Inside thumbnail only show:
- Duration
- View count
- NEW badge if applicable
- Live badge if applicable

Do NOT show these over the thumbnail:
- Large title text
- Username
- Game name
- Captions
- Descriptions
- Engagement row

Under the thumbnail show:
- Title
- Username
- Game tag

Game tag style:
- Neon green background: #B7FF1A
- Dark text: #03080A
- Small pill style
- Rounded corners
- Compact spacing

Clip-specific requirements:
- Clips must remain 16:9 horizontal thumbnails
- Do not convert clips into vertical reel thumbnails
- Preserve aspect ratio
- Keep gameplay visible and uncluttered

Reel-specific requirements:
- Reels stay vertical 9:16
- Keep current Reels card style
- Use it as the reference design

Card styling:
- Thumbnail and metadata should feel like one cohesive card
- Reduce spacing between thumbnail and metadata
- Use consistent border radius
- Add subtle dark background or spacing if needed
- Keep the design clean and premium
- Use Gamefolio theme:
  - Background: #03080A
  - Surface: #0B1218
  - Border: #1B2A33
  - Neon green: #B7FF1A
  - Text primary: #F5F7F2
  - Text secondary: #B8C0AE
  - Font: Space Grotesk

Important:
- Do not change functionality
- Do not change upload/view/like/share behaviour
- Only standardise thumbnail/card layout and styling
- Make thumbnails consistent across the entire app
- Ensure individual user Gamefolio pages match the same thumbnail style
